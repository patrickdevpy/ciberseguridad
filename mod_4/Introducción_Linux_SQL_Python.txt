Como mencionamos anteriormente, las organizaciones utilizan una variedad de herramientas, como los SIEM, los manuales de estrategias y 
los rastreadores de paquetes para administrar monitorear y analizar mejor las amenazas de seguridad. 
Pero esas no son las únicas herramientas del conjunto de herramientas de un analista. 
Los analistas también utilizan lenguajes de programación y sistemas operativos para realizar tareas esenciales.

Te presentaremos la programación en Python y SQL y el sistema operativo Linux. Todo lo cual tendrás la oportunidad de practicar 
usándolo más adelante en el programa de certificación.

Las organizaciones pueden usar la programación para crear un conjunto específico de instrucciones para que una computadora ejecute tareas. 
La programación permite a los analistas completar tareas y procesos repetitivos con un alto grado de precisión y eficiencia. 
También ayuda a reducir el riesgo de errores humanos y puede ahorrar horas o días en comparación con la realización manual del trabajo. 
Ahora que sabe para qué se utilizan los lenguajes de programación, analicemos un sistema operativo específico y relacionado llamado Linux 
y dos lenguajes de programación: SQL y Python.

Linux es un sistema operativo de código abierto o disponible públicamente. A diferencia de otros sistemas operativos con los que puede 
estar familiarizado, por ejemplo macOS o Windows , Linux se basa en una línea de comandos como interfaz de usuario principal. 
Linux en sí mismo no es un lenguaje de programación , pero sí permite el uso de comandos basados en texto entre el usuario y el sistema 
operativo. Encontrará más información sobre Linux más adelante en el programa. Un uso habitual de Linux para los analistas de seguridad 
de nivel básico es examinar los registros para comprender mejor lo que ocurre en un sistema. Por ejemplo, es posible que utilice comandos 
para revisar un registro de errores al investigar un tráfico de red inusualmente alto.

A continuación, analicemos SQL. SQL son las siglas de Lenguaje de Consulta Estructurado . SQL es un lenguaje de programación que se utiliza
para crear interactuar y solicitar información de una base de datos. Una base de datos es una colección organizada de información o datos. 
Puede haber millones de puntos de datos en una base de datos. Por lo tanto, un analista de seguridad de nivel básico utilizaría SQL para 
filtrar los puntos de datos y recuperar información específica.

El último lenguaje de programación que presentaremos es Python. Los profesionales de seguridad pueden usar Python para realizar tareas 
repetitivas y lentas y que requieren un alto nivel de detalle y precisión.

Como futuro analista, es importante entender que el conjunto de herramientas de cada organización puede ser algo diferente en función de 
sus necesidades de seguridad. El punto principal es que estás familiarizado con algunas herramientas estándar de la industria porque eso 
demostrará a los empleadores que tienes la capacidad de aprender a usar sus herramientas para proteger a la organización y a las personas 
a las que sirve.

estas herramientas en situaciones relacionadas con la seguridad.
